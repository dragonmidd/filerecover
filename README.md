# FileRecover (Windows 10 x64)

轻量级文件恢复工具项目骨架（原型）。

目标：恢复误删除或丢失的各类文件，支持 NTFS/FAT/exFAT，提供图形界面与命令行接口。

技术选型（建议）：
- GUI/平台集成：C# (.NET 7/8) + WPF 或 WinUI
- 恢复引擎：C++17/20（编译为 DLL），负责磁盘访问、文件系统解析与文件雕刻
- 测试：xUnit (C#), GoogleTest (C++)
- 打包：WiX / Inno Setup（MSI/安装程序）

项目结构（初始）：
- src/engine/    # C++ 恢复核心
- src/app/       # C# GUI / CLI
- tests/         # 单元与集成测试
- docs/          # 需求与设计文档
